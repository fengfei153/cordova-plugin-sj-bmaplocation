# cordova-plugin-wsr-bmaplocation
百度地图定位Cordova插件

## Installation

    cordova plugin add https://github.com/fengfei153/cordova-plugin-wsr-bmaplocation.git --variable ANDROID_KEY=xxxxxxxxxx

## Methods

- navigator.geolocation.getCurrentPosition

### Supported Platforms

- Android
